Configuración de la Auditoría

Seleccione los objetivos de la auditoría para el análisis de la base de datos:

Verificar contraseñas, permisos y políticas de acceso a la base de datos.

Analizar índices existentes y consultas lentas para optimizar el tiempo de respuesta.

Evaluar el cumplimiento de las formas normales (1FN, 2FN, 3FN) para evitar redundancias.

Verificar si la base de datos cumple estándares como ISO 27001 o PCI DSS.

Nota: Los resultados de la auditoría proporcionarán recomendaciones para mejorar la seguridad y el rendimiento de su base de datos basándose en las opciones seleccionadas.
